# Proyecto_Titanic
## 🚢 Análisis de Datos - Titanic Dataset

Este proyecto consiste en un análisis estadístico realizado en **Python** utilizando **Google Colab** y la librería **Pandas**. El objetivo es extraer información clave sobre los pasajeros y sobrevivientes del naufragio del Titanic.
